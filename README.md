# 🛡️ Agent Guard

AI Agent 安全护栏服务 - 让企业放心使用 AI Agent

## 📌 项目简介

Agent Guard 是一个专为 AI Agent 设计的安全监控服务，提供：
- 权限边界监控
- 敏感数据检测
- 行为日志审计
- 异常告警通知

## 🚀 功能特性

- [ ] 权限边界监控
- [ ] 敏感数据检测
- [ ] 行为日志审计
- [ ] 异常告警系统
- [ ] 安全策略配置

## 💻 技术栈

- 前端：React + Tailwind CSS
- 后端：Python FastAPI
- 数据库：PostgreSQL + Redis
- AI框架：LangChain
